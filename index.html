WebAssembly

Tai yra kursinio darbo projektas kuris panaudoja naują Web technologiją - WebAssembly. WebAssembly yra binarinis kodas, kuris gali būti vykdomas žiniatinklio programose, suteikiantis greičio pranašumą už įprastinę JavaScript kalbą. Darbe yra sukurta C++ programa, kuri naudoja grafikos piešimo biblioteką OpenGL ir programa konvertuojama į WebAssembly formatą, kuris užkraunamas kaip internetinis puslapis.
  • Demonstracinis puslapis: https://arvkus.github.io/webassembly/index.html
